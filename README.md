# react-app-crud

## About the project

This is a simple React web application, along with styled-components and react-bootstrap.

### Features

- [x] Customer registration
- [x] Customer listing
- [x] Customer delete 
- [ ] Customer Update

## How to Install

* To download the project follow the instructions bellow:

```
 git clone https://github.com/pedrovitorrs/react-app-crud.git
 cd react-app-crud
```

* Install the dependencies and start the project:

```
 yarn install
 yarn start
```

or

```
 npm install
 npm start
```
